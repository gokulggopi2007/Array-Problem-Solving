# Array-Problem-Solving

1. write a C program to print the array.
2. write a C program to print sum of the array element.
3. write a C program to print even number in the array.
4. write a C program to print odd number in the array.
5. write a c program to print the sum of even number and odd number in array.
6. write a C program to print largest element in an array.
7. write a C program to print smallest element in an array.
8. write a C program to find the average of array elements.
9. write a C program to print the array in Reverse order.
10. write a C program to find the alternate sum of an array (arr[0] - arr[1] + arr[2] - arr[3] + …).
